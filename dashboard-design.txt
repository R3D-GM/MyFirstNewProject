KEY METRICS IDENTIFIED:
1. Total Customers = Count all records
2. Active Customers = Count where isActive = true
3. Inactive Customers = Count where isActive = false
4. Companies = Count where isPerson = false
5. Persons = Count where isPerson = true
6. Active Rate = (Active / Total) * 100
7. Business Type Distribution = Group by BusinessType

API DATA STRUCTURE:
- id: Unique customer identifier
- code: Customer code
- firstName, secondName, thirdName: Name fields
- isPerson: True = Person, False = Company
- isActive: True = Active, False = Inactive
- businessType: Business category
- gender: Gender
- createdOn: Creation date
- lastModified: Last update date

DASHBOARD SECTIONS:

3. Doughnut Chart - Status distribution (Active vs Inactive)
4. Bar Chart - Analytics comparison
5. Quick Stats - Summary statistics
